Domanda 3

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Un CDN è una rete di server distribuiti geograficamente che fornisce contenuti web agli utenti in base alla loro posizione geografica.

I vantaggi principali rispetto a scaricare Bootstrap localmente sono:
  • Velocità di Caricamento: I contenuti vengono forniti dal server più vicino all'utente, riducendo i tempi di latenza.
  • Sicurezza: Alcuni CDN offrono funzioni di sicurezza integrate, come protezione DDoS e certificati SSL.
Domanda 4

A cosa serve questo attributo del tag meta? "content="width=device-width, initial-scale=1"

L'attributo content="width=device-width, initial-scale=1" del tag serve a controllare il comportamento della pagina web sui dispositivi mobili.
